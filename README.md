# rubykoans *(working title: Ruby Path)*

A modern, Rustlings-inspired learning tool for Ruby 4+. Replaces the legacy Edgecase Ruby Koans curriculum.

> Status: pre-release. Phase 1 of 6 — walking skeleton.

## Quick start (Phase 1, local install only)

```sh
bundle install
gem build rubykoans.gemspec
gem install ./rubykoans-0.1.0.gem
koans init
koans run
```

Detailed docs land in Phase 6.
